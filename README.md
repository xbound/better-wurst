How to use:

Select a branch except main.

Download the source code from https://github.com/Wurst-Imperium/Wurst7

Replace these file:

  src/main/java/net/wurstclient/hacks/...Hack.java
  
  src/main/java/net/wurstclient/util/ChunkSearcher.java

Build
```
  ./gradlew genSources
  ./gradlew build
```
